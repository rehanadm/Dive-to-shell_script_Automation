Shell Script Automation for Linux system administration tasks (health check, log rotation, cleanup). Designed for DevOps and production environments.

# Shell Script Automation

## 📌 Overview
This project contains shell scripts to automate daily Linux administration tasks.

## 🚀 Scripts

### 🔹 Health Check Script
- CPU usage
- Memory usage
- Disk usage
- Service status

### 🔹 Log Rotate Script
- Rotates logs
- Prevents disk overflow

### 🔹 Delete Old Logs
- Deletes logs older than X days

## ⚙️ Usage
```bash
chmod +x script.sh
./script.sh
